Java 三大特性 封装 继承 多态

封装

对属性、方法、类、项目进行封装

对属性进行封装

  1. 对成员变量加修饰符private
  2. 使用setAttribute方法来添加属性
  3. 使用getAttribute方法来获得属性

This 关键字

This在类中，代表当前对象

this关键字可以调用属性和方法

this这个关键字只能在方法中使用

This还可以调用当前的构造方法（少用）

使用注意事项

  • 调用成员变量。在构造方法，在成员方法
  • 调用成员方法。在构造方法，在成员方法

this()可以调用构造方法，但只能在构造方法中调用构造方法，且只能写在第一行。

类对象作为方法的参数

方法可以改变对象参数的状态

方法不能修改基本数据类型的参数
